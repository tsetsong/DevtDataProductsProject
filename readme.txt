The work in this repository is done in fulfillment of the Week 4 project requirements of
the Developing Data Products module of the Coursera Data Science Specialization course.

This directory contains 2 files for a shiny app that has been deployed at
https://tsetsong.shinyapps.io/Project/.

ui.R contains R code for user interface elements
server.r contains server side processing and input/output code for prediction and plotting of graphs.
